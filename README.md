# 广西高考志愿助手 GaokaoGX

专为 **广西2026物理类考生** 设计的纯前端高考志愿填报辅助工具。

## 功能

### 🎯 冲稳保推荐引擎（你的模型）
- 基于**位次法**的智能推荐算法
- 输入分数和位次，自动生成 冲/稳/保 三档院校推荐
- 分 **定向师范（提前批）** 和 **工科（本科批）** 两个方向
- 基于科目能力的专业匹配建议
- **纯前端离线可用，无需网络**

### 🤖 AI 角色对话
- 内置广西志愿顾问、定向师范专家、工科指导师三个角色
- 连接 AI API（DeepSeek/GLM）后可进行对话咨询
- 角色已预置广西2026数据和你的考生背景

### 📝 数据自定义
- 可自由修改分数、位次、各科成绩
- 数据保存在浏览器本地

## 部署到 GitHub Pages

1. Fork 或上传本项目到你的 GitHub 仓库
2. 进入仓库 Settings → Pages
3. 选择 `main` 分支，`/` (root) 目录
4. 点击 Save，等待几分钟
5. 访问 `https://你的用户名.github.io/gaokao-guangxi/`

或直接用 VSCode 的 Live Server 打开 `index.html` 本地预览。

## 数据说明

- 分数线和院校数据基于广西招生考试院2025年公开数据
- 推荐算法基于位次法（冲稳保）
- **正式填报请以广西招生考试院官网 (gxeea.cn) 公布的最新数据为准**

## 与原项目的区别

基于 [TsangHaotian/GaokaoHelper](https://github.com/TsangHaotian/GaokaoHelper) 设计理念改进：

| 特性 | 原项目 | 本工具 |
|------|--------|--------|
| 定位 | 通用AI对话 | 广西专用 + 数据引擎 |
| 推荐算法 | 无（依赖AI生成） | 内置位次法引擎 ✅ |
| 离线可用 | 否（需要API） | 推荐引擎完全离线 ✅ |
| 广西数据 | 无 | 内置2025录取数据 ✅ |
| 科目能力分析 | 无 | 基于成绩推荐专业 ✅ |
